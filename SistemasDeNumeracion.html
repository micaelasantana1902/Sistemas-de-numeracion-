Sistema Decimal

El sistema más utilizado del mundo

¿De dónde viene?
Civilizaciones antiguas contaban con base 10, probablemente porque los humanos tenemos diez dedos.
El sistema decimal moderno surgió en la Antigua India, aproximadamente en los siglos V y VI d.C.
Allí se desarrolló algo fundamental: el uso del 0 como número y el sistema posicional que usamos hoy.
Ese fue un cambio enorme: permitió escribir los números de manera mucho más sencilla.
Usamos el decimal para...

Medir, contar y hacer cálculos en la vida cotidiana.

Lo usamos en todo momento: al comprar, cocinar, medir distancias, calcular el tiempo y mucho más.

Sin el sistema decimal, las matemáticas como las conocemos hoy no existirían.

Sistema Binario

¿Qué es el sistema binario?

Es un sistema de numeración que usa un conjunto de reglas para expresar números empleando solo dos símbolos llamados cifras o dígitos. La cantidad de símbolos determina la base del sistema — por eso el binario es base 2.
¿Cómo leer un número binario?

Debés pronunciar cada dígito por separado. Por ejemplo: 101 Se lee "uno cero uno" — no "ciento uno".

Así no se confunde con el número decimal ciento uno (101).

Conversión

Pasar de Binario → Decimal

Cada dígito del número binario tiene un valor que depende de su posición. Lo que hay que hacer es sumar esos valores, leyendo de derecha a izquierda.
¿Cómo se hace?
1
Cada posición = potencia de 2
Una potencia es multiplicación repetida. Ej: 2×2×2×2 = 2⁴
2
De derecha a izquierda
El primer dígito a la derecha tiene la potencia 2⁰ = 1
3
Cada posición suma 1 a la potencia
2do: 2¹, 3ro: 2², 4to: 2³ y así sucesivamente
4
Multiplicar y sumar
Si el dígito es 1, suma el valor. Si es 0, no suma nada.
